Bạn đã hiểu cách tạo snippet

Nếu chưa hiểu thì xem lại video hoặc comment hỏi ở bên cạnh.

  • Copy đoạn code cần lưu vào snippet
  • Tool > new snippet
  • paste code vao CDATA
  • dịnh nghĩa kí tự gợi nhớ
  • định nghĩa giới hạn sử dụng bằng scope.
  • Lưu lại đuôi .sublime-snippet
  • Hưởng thụ thành quả !
